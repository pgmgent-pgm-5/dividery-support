Adding Group bills

To split expenses with others in DiviDery, you must first add a bill to a DiviDery Group.

To add a bill to a Group

  1. Click the Groups icon on the menu bar.

    The Groups tab opens.

  2. Click a Group.

    The Group’s details display.

  3. Click Add a bill.

    The Create a bill form displays.

  4. Enter a name for the bill in the Event Title text box, and a monetary amount in the Amount text box.
  5. Select the bill category from the Select a Category dropdown menu.
  6. Select an option from the Splitting Method dropdown menu.
    If Then
    You want to split costs evenly Select the Equal method
    You want to decide the portion of a bill each member pays Select the Percentage method
  7. Click Create to finalize the bill.

    The bill is added to the Group details.

You have successfully created a Group bill! Group members receive an automatic email notification that they have a new outstanding debt. Member balances automatically adjust to reflect the new debt.

If you need to cancel a bill, reopen the Group details and click Delete Bill next to the bill you want to cancel.
