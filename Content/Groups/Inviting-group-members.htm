Inviting Group members

After you create a Group, you can invite other people to become Group members.

To invite someone to a Group

  1. Click + Add a user.

    The Add a user form displays.

  2. Do one of the following:

    Choose a Contact from the Select a user dropdown menu.

    Enter a First name, Last name, and Email in the text boxes.

  3. Do one of the following:

    Click Add if you selected an existing Contact.

    Click Create User if you entered new Contact information.

    The Send Link form displays.

  4. Click Send to choose an invite option.

    DiviDery sends the invite link to the Contact’s email.

    If Then
    You want the new Group member to add and remove bills Choose “Everyone with this link can add expenses to DiviDery”.
    You want the new Group member only view bills without making changes Choose “Everyone with this link can see the DiviDery but cannot make any changes or add expenses.”

Your new Group members will appear in the Group details when they accept the invitation by clicking the invite link in their email.
