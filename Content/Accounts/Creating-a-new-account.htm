Creating a new Account

Create a new Account by accessing the the DiviDery website.

To create an Account

  1. Click Register on the DiviDery home page.

    The Register form displays.

  2. Enter your First name, Last name, Email, and a password in the text boxes.

    Note: If you already have a DiviDery Account, click Log In below the form.

  3. Enter an image link the Avatar URL text box to give your profile an avatar.

    Note: The easiest way to do this is to right-click an image you like on the internet and copy the image link address. Then simply paste the copied link into the text box. Use a square image for best results!

  4. Click Register.

    The DiviDery dashboard opens. A confirmation message will also be sent to the email you provided.

You’re now ready to create a Group and start splitting bills!

Profile settings

Profile settings include your user name, email, password, and avatar.

Profile setting Description
User name The name you go by on DiviDery. This is the name all your contacts and group members will see.
Email The email address DiviDery will send all notifications and reminders to. You also need this email address to recover your password if you forget it.
Password Your password for logging in to your DiviDery account. Be sure to use a strong password with at least 8 characters, one number, and one special character.
Avatar A small image that represents you to your contacts and group members on DiviDery. You must link to this image with a URL. You cannot upload a local file from your device.
