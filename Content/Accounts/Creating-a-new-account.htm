Creating a new Account

Create a new Account by accessing the the DiviDery website.

To create an Account

  1. Click Register on the DiviDery home page.

    The Register form displays.

  2. Enter your First name, Last name, Email, and a password in the text boxes.

    If you already have a DiviDery Account, click Log In below the form.

  3. Enter an image link the Avatar URL text box to give your profile an avatar.

    The easiest way to do this is to right-click an image you like on the internet and copy the image link address. Then simply paste the copied link into the text box. Use a square image for best results!

  4. Click Register.

    The DiviDery dashboard opens. A confirmation message will also be sent to the email you provided.

You’re now ready to create a Group and start splitting bills!
